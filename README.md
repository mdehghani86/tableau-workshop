# Tableau Workshop — IE 5374 Storytelling with Data

A three-session hands-on Tableau workshop for IE 5374 at Northeastern University. Students build their first dashboards from raw data, explore visual analytics (marks, hierarchies, parameters, sets), and learn how to tell a story with data.

**Instructor:** Prof. Mohammad Dehghani

## Interactive outline

Open **[`index.html`](./index.html)** in a browser. It contains:

- All task instructions (Session 1 fully built, Sessions 2 & 3 as stubs)
- Direct download links for every dataset
- Progress checkboxes saved in localStorage
- Quiz callouts

## How students use it

1. Clone or download this repo (green **Code → Download ZIP** button).
2. Open `index.html` in any browser.
3. Download the dataset for the current session from the cards at the top.
4. Follow the task list — check each box as you complete it.

## Tools

- **Tableau Public** (free) or **Tableau Desktop** (student license)
- Any modern browser for the outline page
